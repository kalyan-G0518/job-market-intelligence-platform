# Job Market Intelligence Platform

## Overview

Built an end-to-end cloud data engineering platform using Python, AWS S3, AWS Glue, Athena, and Power BI.

## Architecture

Kaggle Datasets
↓
Python ETL
↓
AWS S3 Data Lake
↓
AWS Glue Catalog
↓
Athena Analytics
↓
Power BI Dashboard

## Dataset

- 9,118 job postings
- 4,329 companies
- 1,455 locations

## Key Insights

- SQL is the most demanded skill.
- AWS appears in 3,179 job postings.
- New York is the top hiring city.
- LinkedIn contributes 66% of job records.

## Technologies

- Python
- Pandas
- AWS S3
